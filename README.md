# Anneke 21 Feest

Deze website is gepubliceerd via MkDocs.

## Lokaal draaien

Start de server met:

```
mkdocs serve
```

Let op: de lokale server draait op het pad `/anneke-21-diner/` (afgeleid van de site_url in mkdocs.yml).
